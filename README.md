# Nutrition Facts App




## How to install

## Usage
- Frontend: http://localhost:3000
- Search foods and view nutrition and portion data. Backend API:
    - GET /api/food/search/best?name=: Returns best-matching food DTO.
    - GET /api/food/portion?fdc_id=: Returns aggregated food portion and unit data.
